# 股票筛选分析系统

这是一个基于Python的股票筛选和分析系统，主要功能包括：

- 股票数据采集：从各数据源获取股票基本信息和财务数据
- 股票筛选：根据财务指标和技术指标筛选股票
- 数据可视化：展示股票数据和分析结果

## 安装依赖

```bash
pip install -r requirements.txt
```

## 项目结构

- `stock_data_collector.py`: 股票数据采集模块
- `stock_screener.py`: 股票筛选模块
- `stock_viewer.py`: 数据展示模块
- `main.py`: 主程序入口

## 使用方法

运行主程序：

```bash
python main.py
```
